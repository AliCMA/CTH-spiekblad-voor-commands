  • Cheatsheet Ali Celiksu Onderzoeksvraag: Hoe zet je vanuit een lege map een repository op Github? De stappen:
      1: Git status: zo laat je de status van je git zien.
      2: Git init: zo begin je een nieuw git repository.
      3: git add . : dan word alles in 1x toegevoedg.
      4: Git log: zo zie je alle veranderingen.
      4: Git --oneline: Alles word hiermee op 1 lijn gezet.
      5: Git push: dan push je het naar Github toe.
      6: Git pull: Dan haal je de veranderingen op.

cmd

  • cd: change directory
  • mkdir: maak een nieuwe directory
  • cd .. : dan ga je 1 stap omhoog
  • cd ../ : dan ga je meerdere stappen omhoog
  • %USERPROFILE% : dan ga je naar je main directory
